Spriki Functions

Microsoft 2003 has 247 functions available to use in formulas. So far 240 have been implemented, but only 100 are available in this version of the Spriki - the rest are awaiting a rewrite of the input 'language' which is being made more compatible with Microsoft Excel. However the spriki component which maps the functions which the user types into formulas to the actual code that runs is not complete - particularly with relation to the acceptance of strings. Therefore whilst these functions are available, not all of them can be called yet :(

There are a number of differences between the way in which Spriki and Excel handle a number of functions. Typically repeated arguments which are handled via arguments in Excel are handled as lists in the Spriki. For instance the Excel function Sum takes between 1 and 30 arguments, eg:
=sum(a1,b3,b5)
In the Spriki this would be written as:
=sum(['x1:y1','x2:y3','x2:y5'])
