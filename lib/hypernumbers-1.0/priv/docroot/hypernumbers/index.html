Welcome to the hypernumbers beta!

Thank you for volunteering to test hypernumbers with us, this guide will give you a brief guide to how to use/test hypernumbers.

Warning: During the beta phase you are advised to backup any data you need locally.

Report Bugs and Feedback

During this beta phase there are still lots of known bugs to be fixed and features to be implemented. If you hit any bugs, or if there is anything you would like to see in hypernumbers, please click the red toolbar button as shown below.

We are upgrading hypernumbers very regularly, In future we will contact you directly to update you with bug fixes and new features, but for now you can follow us on twitter to get updates @ twitter.com/hypernumbers

Spreadsheets as Urls

Hypernumbers allows you to organise your spreadsheets in a flexible way similiar to a file system. Under the My Spreadsheets menu you can create new spreadsheets and view existing ones, including publicly shared spreadsheets.

Using data from other spreadsheets is easy, at the bottom right you will see your currently selected cell displayed as a formula that you can enter in any other sheet. just copy and paste the text into a cell and you are done
