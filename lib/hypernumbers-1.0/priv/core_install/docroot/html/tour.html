Spreadsheets as Urls

Hypernumbers allows you to organise your spreadsheets in a flexible way similiar to a file system. Under the My Spreadsheets menu you can create new spreadsheets and view existing ones, including publicly shared spreadsheets.



You can use these url addresses when linking to cells on other spreadsheets, you can find out more about that on the next page.

HyperNumbers

Using data from other spreadsheets is easy, select the cells you are interested and copy them using: right click > Copy, edit menu > Copy or Ctrl+C

Then select where you want the cells to go, and use "Paste Links" from the edit menu or right click menu. The data will update as if it were a link on the same spreadsheet.
